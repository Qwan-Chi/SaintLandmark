# SaintLandmark 🏛️

Гид по достопримечательностям Санкт-Петербурга. Приложение предзагружено каталогом из 5 знаменитых мест (Эрмитаж, Петропавловская крепость, Спас на Крови, Новая Голландия, Петергоф) с описаниями, фотографиями и возможностью отмечать избранное.

## Возможности

- 📋 Каталог достопримечательностей с фотографиями и описаниями
- 🖼️ Галерея изображений с бесконечной прокруткой (HorizontalPager)
- ⭐ Избранное — отметка мест звёздочкой + отдельный экран избранного
- 💬 Комментарии к каждому месту (сохраняются в БД)
- 🎨 Шрифт IBM Plex Serif
- 🌐 Загрузка фото через интернет (Coil)

## Технологии

| Категория | Технологии |
|-----------|-----------|
| Язык | **Kotlin** 1.8.10 |
| UI | **Jetpack Compose** + **Material 3** |
| База данных | **JetBrains Exposed** 0.45.0 + **H2** (file DB) |
| Навигация | **Navigation Compose** 2.7.5 |
| Изображения | **Coil** (coil-compose 2.5.0) |
| Шрифты | Google Fonts (IBM Plex Serif) |
| Min SDK | 31 (Android 12+) |
| Target SDK | 34 |

## Запуск

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/Qwan-Chi/SaintLandmark.git
   ```
2. Открой в **Android Studio**
3. Дождись Gradle sync
4. Запусти на устройстве/эмуляторе (API ≥ 31)

```bash
./gradlew assembleDebug
```

> ℹ️ Приложению нужен доступ в интернет для загрузки фотографий достопримечательностей.

## Структура проекта

```
app/src/main/java/com/example/kurilichevproject/
├── MainActivity.kt            # Точка входа, стартовый экран
├── OverViewScreen.kt          # Каталог мест (карточки)
├── InfoView.kt                # Детали: галерея, избранное, комментарий
├── Favorites.kt               # Экран избранного
├── db/                        # Exposed: таблицы, подключение к H2
├── navigation/                # NavHost + инициализация БД (5 мест)
└── ui/theme/                  # Material 3 тема
```
